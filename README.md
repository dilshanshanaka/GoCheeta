# GoCheeta
BSc (Hons) in Software Engineering - CIS6003  Advanced Programming (Coursework - Online cab service system)

GoCheeta is a web application that can be accessed via the web browser to manage cab (vehicle) bookings online. GoCheeta web application customers need to register as a customer to use the service. After registering to the system customer can place a booking request by selecting an area where the GoCheeta branch is located. Customers will be prompted with pickup and destination options to plan their journey. Then customers can select a vehicle type and place a booking. Drivers can accept bookings. Admin can manage users, and vehicles and view earnings.
